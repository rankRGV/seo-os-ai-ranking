# Data Policy

This repository is a public/community template. It must not contain internal business data or client data.

## Allowed

- Fake demo clients
- Fake metrics
- Generic workflow examples
- Empty template folders created by `.gitkeep`
- Documentation

## Not allowed

- Real analytics exports
- Real Search Console exports
- Real Google Sheet IDs
- Real Discord channel IDs or thread IDs
- OAuth tokens or credentials
- Real client names/domains unless intentionally public and approved
- Local SQLite databases
- Logs and generated artifacts
